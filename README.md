# 📖 Project Overview

Supply chain efficiency directly impacts business profitability, inventory management, supplier relationships, and customer satisfaction.

This project analyzes supply chain operations data to identify:

✔ Revenue Drivers  
✔ Product Performance  
✔ Supplier Performance  
✔ Location-Based Revenue Trends  
✔ Transportation Insights  
✔ Inventory & Lead Time Metrics

The analysis was performed using:

- Excel
- MySQL
- Python (Pandas)
- Matplotlib
- Google Looker Studio

---

# 🎯 Business Problem

Organizations need visibility into:

- Which products generate the highest revenue?
- Which suppliers contribute the most value?
- Which locations perform best?
- How inventory levels impact operations?
- Which transportation modes are most effective?

This dashboard provides actionable insights to support data-driven decisions.

# 🔍 Key Business Insights

### Product Analysis
- Skincare products generated maximum revenue.
- Product sales distribution shows strong category concentration.

### Supplier Analysis
- Top-performing suppliers drive a large percentage of total revenue.

### Geographic Analysis
- Mumbai and Kolkata are the strongest revenue-generating markets.

### Operational Analysis
- Inventory levels remain stable.
- Lead times suggest opportunities for optimization.
